# Welcome To Vocabulo

Welcome to Vocabulo! Your software to help you learn new words easily with better UI than a regular notepad. Developed using Expo, React Native and Supabase as the cloud database

![Vocabulo App Images](./Images/Logo.jpg)

## Set Up Guide

1. Clone the project
2. Open the project in Visual studio and make sure you are in the right directory.
3. Run `npm install`.
   
## Run The Project

Now that the project is set up, it is ready to be used
1. Run the project with `npx expo start`. You could also use commands like `npx expo start --web` for website development environment directly. Complete list of script commands can be found in package.json

## Building the project 

To build an app for android or IOS, you can use the free eas services provided by expo and the information can be found on their **[website](https://docs.expo.dev/build/setup/)**.
